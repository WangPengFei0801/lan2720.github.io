什么是Java当中的软件包?

所谓的软件包
: 就是把类放在不同的文件夹下

相当于我们平时使用电脑时，有两个同名的文件，但是这两个文件我们都想保存，那么放在不同名的文件夹下，就可以了。对于Java软件包，使用的是同一个原理。

同时，可以理解为java的软件包
: 提供了命名空间

即规定了两个类可以重名，但是是两个不同的空间内可以重名。

为什么要用软件包

举个栗子，某公司对学校的管理系统进行开发，A组人员创建了一个父类User，下面有两个子类Student和Teacher。由于是单独开发，B组人员并不知道A组人员的命名，他们也定义了一个User父类，有RegUser和UnRegUser两个子类。

A组开发 B组开发
User User
Student RegUser
Teacher UnRegUser

开发完成后，各自的模块都测试正常，但是合并成完整系统时，问题出现了——存在两个User类。怎么办?
这个时候，Java通过软件包完美解决了这个问题。
